# HR Analytics Report – Attendance & Work Mode Analysis

## Project Overview
This project is an end-to-end HR analytics analysis focused on understanding employee attendance behavior, work mode preferences, and leave patterns.  
The goal is to help HR and operations teams gain insights into workforce availability and support data-driven decisions related to hybrid work policies and attendance planning.

---

## Problem Statement
Organizations need visibility into how employees attend work, utilize leave, and choose between work-from-office and work-from-home.  
This project aims to identify attendance trends, work mode patterns, and leave distribution over time to support effective HR operations and workforce planning.

---

## Dataset Description
The dataset contains employee-level attendance records captured over a period of time.

### Key Data Fields
- Employee ID
- Date
- Attendance Status (Present / Absent / Leave)
- Work Mode (Work From Home / Work From Office)
- Leave Type (Sick Leave, Casual Leave, Paid Leave)

---

## Tools Used
- Power BI
- Power Query
- DAX

---

## Analysis Process

### 1. Data Cleaning & Preparation
- Imported raw HR attendance data into Power BI
- Cleaned and standardized attendance and work mode values
- Converted date fields to appropriate formats
- Created calculated columns and measures for attendance and leave metrics

---

### 2. Data Analysis
The analysis focused on identifying patterns related to:
- Employee attendance trends across time
- Work-from-home vs work-from-office preferences
- Leave behavior and leave type distribution

Key observations include:
- Attendance levels vary by day and over time
- Work-from-home usage shows clear patterns across the week
- Sick leave and other leave types contribute differently to overall absence

---

### 3. Dashboard Creation
Interactive dashboards were built to communicate insights clearly to stakeholders.

#### Dashboard 1: Attendance & Work Mode Overview
- Attendance percentage over time
- Work From Home vs Work From Office distribution
- Overall workforce availability trends

#### Dashboard 2: Leave Analysis
- Sick Leave percentage
- Casual and Paid Leave distribution
- Leave trends over time

---

## Key Insights
- Employee attendance shows noticeable variation across different days
- Work-from-home adoption follows consistent weekly patterns
- Sick leave contributes a measurable portion of overall absence
- Attendance and work mode insights can support better workforce planning

---

## Outcome
This project demonstrates an end-to-end HR analytics workflow using Power BI — from raw data preparation to insight-driven dashboards.  
The analysis provides actionable insights that can help organizations optimize attendance policies and manage hybrid work environments more effectively.
